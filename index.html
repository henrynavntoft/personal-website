Projects

Beerwalks Copenahgen

One of my projects was the Beerwalks Copenhagen website. Using semantic HTML and CSS, I created an engaging and accessible experience for visitors. I also implemented JavaScript to enhance the website's interactivity and functionality. The design effectively promotes Beerwalks Copenhagen as a unique experience for beer enthusiasts.

Copenhagen Cartel

Another project I created was a landing page for Copenhagen Cartel. We needed to make a landing page aimed at engaging a younger audience. Using HTML, CSS, and JavaScript, I crafted a visually appealing and dynamic website that captures the attention of the target audience.

Hacked Hogwarts

Lastly, I developed Hacked Hogwarts, a web application that simulates a cyber attack on Hogwarts School of Witchcraft and Wizardry. Through basic JavaScript techniques such as filtering and sorting, I developed a user interface that enables users to interact with the simulated cyber attack. This project helped me enhance my JavaScript programming skills and gain experience in creating dynamic web applications.
